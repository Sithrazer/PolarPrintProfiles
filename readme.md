Polar Filament Printing Profiles

These are user submitted tuned profiles for Polar Filaments.

When submitting a profile it should be using the format: 
Printer.Nozzle_Size.Filament_Type.3mf

A PLA profile for a Bambu Lab X1C with a .4mm nozzle the filename would look like:
X1C.4.PLA.3mf

The 3mf file should contain a Benchy that is created by right clicking and adding a Benchy from your slicer of choice.

DO NOT Submit tuned settings for the printer. Just a tuned filament profile.
